# 🌐 Desenvolvimento Web - 5º Semestre | UNINOVE

Este repositório contém o material prático, exercícios e projetos desenvolvidos durante a disciplina de **Desenvolvimento Web** do 5º semestre na **Universidade Nove de Julho (UNINOVE)**.

O foco principal é o aprofundamento em tecnologias Front-end e a criação de interfaces responsivas e dinâmicas.

---

## 🚀 Tecnologias Utilizadas

As aulas e projetos deste semestre focam no "Core" do desenvolvimento web moderno:

*   **HTML5:** Semântica, acessibilidade e estruturação avançada.
*   **CSS3:** Layouts com Flexbox, CSS Grid, animações e design responsivo (Mobile First).
*   **JavaScript (ES6+):** Manipulação do DOM, eventos, Promises e consumo de APIs.
